Project Overview

This website is the official product catalog and ordering platform for SutiYam, a fashion brand offering men's and women's formalwear, accessories, and custom clothing.

Project Structure

  • index.html - Main landing page
  • OurCollection_Updated.html - Product listings (with sorting, images, and pricing)
  • checkout.html - Order form with email submission via FormSubmit
  • returns-policy.html - Return & refund information
  • delivery-policy.html - Delivery timelines and info
  • sitemap.xml - SEO sitemap for search engines
  • WeThrive.js - JavaScript for cart logic, sorting, and interactivity

Features

  • Add-to-cart system (with localStorage)
  • Checkout form with automatic order summary and auto-email
  • Fast loading with optimized images
  • Hosted on Netlify
  • Sitemap and SEO-ready
